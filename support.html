Support

Help for VELA import, transcription, subtitle editing, translation, preview, and export workflows.

Last updated: 2026-04-30

Contact

For support requests, please email msk00000@kakao.com.

When contacting support, include the app version, iPad model, iPadOS version, what workflow you were using, and whether the issue happened during import, transcription, translation, review, or export.

Device Requirements

VELA is designed for iPad. Local AI model execution requires an M-series iPad or an iPad with sufficient memory. If tested on unsupported hardware, the app may show an unsupported-device screen.

Import Help

  • Use Files or Photos to choose media through iPadOS system UI.
  • Confirm that the media file is still available at the selected location.
  • If a permission prompt appears, grant access only when you want VELA to process that file.

Transcription Help

  • Apple Speech transcription may require Speech Recognition permission.
  • Optional Whisper models must be installed before Whisper-based transcription can run.
  • Long media files and large models may require additional time and device storage.

Translation and Local Model Help

  • Optional local language models are downloaded only when selected by the user.
  • Model downloads may be several gigabytes and require stable network connectivity.
  • Generated translations may contain errors and should be reviewed before use.

Export Help

  • Exports are saved through user-selected iPadOS save or share locations.
  • Review subtitles before publishing or distributing exported files.
  • If export fails, try a shorter media file or a subtitle-only export format first.
