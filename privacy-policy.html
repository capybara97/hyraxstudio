Privacy Policy

VELA helps users transcribe imported media and prepare editable subtitles on iPad.

Last updated: 2026-04-30

Overview

VELA is an iPad-only on-device media transcription, subtitle editing, and translation workflow app.

Media files are selected by the user through the system file picker, photo picker, or other iPadOS-provided import surfaces. VELA processes imported media for transcription, subtitle editing, translation, preview, and export workflows.

On-Device Processing

VELA is designed around on-device processing. VELA does not upload the user's imported media, audio, subtitle text, or translation text to a VELA-owned server.

Generated transcripts, subtitles, translations, previews, and exports are created as part of the user's local workflow unless the user chooses an iPadOS share or save destination.

System Permissions

When the user chooses Apple Speech transcription, iPadOS may require Speech Recognition permission and may process speech according to Apple's system speech recognition behavior and user settings. Users can manage this permission in iPadOS Settings.

When the user imports media from Files, Photos, or another system source, file access is initiated by the user through iPadOS-provided selection interfaces.

Optional Model Downloads

Optional on-device Whisper or local LLM model files may be downloaded from external model hosting providers only when the user chooses to install those models.

These downloads are model files used by the app and are not uploads of user media, subtitle content, or translation content. Model files may be large and may require sufficient device storage, memory, and network connectivity.

Exports and Sharing

Exported subtitle or video files are saved or moved only through user-selected iPadOS save or share locations. The destination is controlled by the user.
